# 6. 数据库概览

`repomix-rs` 不使用任何数据库。所有状态存在三个地方：

- **内存**：`Vec<RawFile>`、`Vec<ProcessedFile>`、`HashMap`指标等。
- **用户配置文件**：`~/.repomix/repomix.config.json` 和当前项目下的 `./repomix.config.json`，两者都通过 `serde_json` 读写，不是 SQL。
- **输出文件**：打包结果写入文件系统上的一个普通文件（XML/Markdown/Plain/JSON）。

因此本仓库不含任何 `.sql` / `.sqlproj`、`migrations/`、`sql/`、`db/`、`database/` 目录，也没有 ORM 依赖或数据库配置文件。如果你需要数据持久化，最自然的存储位置是一个新的 Rails / Diesel / SQLx 后端，但这超出了当前项目的边界。
